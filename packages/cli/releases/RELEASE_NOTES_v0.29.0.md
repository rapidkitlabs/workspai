# RapidKit v0.29.0 Release Notes

### 🧪 Workspace Snapshot Support and Docs Cleanup

This release delivers workspace snapshot support along with repository and documentation cleanup.

**What's New:**

- 🗂️ **Workspace snapshot support**
  - Added workspace snapshot manifest support for reliable workspace state inspection and archive workflows.
  - Added support for metadata-only and full workspace snapshot modes.

- 🧾 **Docs and release metadata cleanup**
  - Updated GitHub repository links and references from `getrapidkit` to `rapidkitlabs` across README and docs.
  - Synced release note references and package metadata.

- ✅ **Stability and coverage**
  - Added regression coverage for workspace snapshot behavior.

## Upgrade

```bash
npm install -g rapidkit@0.29.0
```

# wspai

Short npm alias for the Workspai CLI.

```bash
npx wspai --help
```

This package delegates to the canonical `workspai` package. Keep the `wspai` package version and its
`workspai` dependency version aligned with `packages/cli/package.json`.

Canonical package:

```bash
npx workspai --help
```
